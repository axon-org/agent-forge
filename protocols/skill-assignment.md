# Skill Assignment Protocol

## Goal
Assign the smallest effective skill set per agent.

## Process
1. Map agent mission to domains.
2. Install only high-signal skills for those domains.
3. Add trigger phrases so skill loading is predictable.
4. Validate no overlap/conflict with existing specialist agents.

## Review Questions
- Is each skill directly tied to recurring tasks?
- Is there a clear “do not use” boundary?
- Can the skill be replaced by a reusable template section?
