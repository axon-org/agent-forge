# Agent Design Philosophy

This repository replaces monolithic playbooks with modular building blocks.

## Principles
- **Specialists over generalists**: each agent has explicit scope.
- **Safety over speed**: irreversible actions require explicit approval.
- **Learn continuously**: error-learning and memory are first-class.
- **Composable docs**: templates + sections instead of one giant file.
- **Verification before claims**: never mark complete without checks.
