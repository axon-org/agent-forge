# Workspace Anatomy

Core files every agent workspace should include:

- `SOUL.md`: identity and behavioral core
- `AGENTS.md`: operational rules and protocols
- `TOOLS.md`: concise tool reference
- `USER.md`: who the agent serves and comms style
- `IDENTITY.md`: stable profile metadata
- `HEARTBEAT.md`: health/cadence snapshot
- `MEMORY.md`: long-term curated memory
- `memory/YYYY-MM-DD.md`: daily execution logs
